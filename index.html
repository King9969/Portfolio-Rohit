My Projects

Smart Wheelchair

This system is designed to assist individuals with disabilities in performing daily tasks using Python, Natural Language Processing, Firebase, IoT, and OpenCV technologies. The system provides accessibility options that allow users to control their computer, surf the internet, operate appliances, and control their wheelchair using their eyes, voice, muscle movements, or gestures. The use of Natural Language Processing technology enhances the system's ability to understand and respond to user commands. Firebase, a cloud-based database, provides reliable storage and retrieval of user data, while IoT technology allows for seamless communication between devices. OpenCV, an open-source computer vision library, is used for image and facial recognition. Overall, this system provides an accessible and intuitive solution for people with disabilities to enhance their mobility and independence.

Smart Helmet with Crash & Wear Detection

This IoT-based solution is designed to enhance bike safety by utilizing a combination of React, API's, MongoDB, and IoT technologies. The system features wear detection, which prevents the bike from starting if the rider is not wearing a helmet. Additionally, the system includes crash detection that sends an emergency notification with the rider's location if a crash occurs. React, a popular JavaScript library, will be used to create a user-friendly interface. API's will enable communication between the various components of the system, and MongoDB will serve as the database for storing rider information and bike data. Overall, this system has the potential to greatly improve bike safety and save lives.
